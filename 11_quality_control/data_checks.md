# Data quality checklist

Use this checklist to keep reporting trust high while the broader CRM model is still being clarified.

## Daily
- Missing lead source?
- Missing campaign?
- Missing referral source?
- Duplicate leads?
- Overdue follow-up?
- Contradictory statuses?

## Weekly
- Lead source coverage drift?
- Campaign coverage drift?
- Referral source coverage drift?
- Pre-active leakage increase?
- Referral aging or stalled follow-up?
- Channel inconsistencies?

## Directional only checks
- Authorized pipeline jump?
- Referral-to-active swing?
- Dropped referral spike?

## Red flags
- Sudden lead drop
- Broken attribution
- Rising overdue follow-up count
- Authorized count being presented as active census
