Decision rules

  1. Compare engines on cost per SOC, not leads or touches alone.
  2. If digital CPL looks great but SOC CPA worsens, treat as lead-quality collapse.
  3. If field activity is high without referral yield, change coverage or coaching—do not celebrate touches.
  4. If referrals are strong but intake → auth → SOC is weak, escalate ops/payer; do not blame the channel blindly.
  5. Reallocate weekly/monthly using stop / continue / double on this sheet.
